# Ashli's Custom Search Engine
## An Assignment for Rutgers' 530 Search & Information Landscape
### Spring 2026

Are you looking for 3D print files? Use [this custom search engine](https://cse.google.com/cse?cx=05dd46c40c842437e).

This search engine includes the following websites:
- www.tinkercad.com/things*
- 3d.si.edu/*
- www.myminifactory.com/*
- nasa3d.arc.nasa.gov/models*
- www.instructables.com*
- *.repables.com/*
- www.yeggi.com/*
- science.nasa.gov/3d-resources/*
- www.creativefabrica.com/3d-printing/*
- www.etsy.com/search?q=3d+files*
- *.cults3d.com/*
- makerworld.com/en*
- www.printables.com/*
- www.thingiverse.com/*

Although several of these sites will repeat themselves in terms of search results, each does provide brand new information in a search. This is intended to make it easier for folks new to 3D printing to be able to point them in the right direction without needing to manually look on several individual sites.
